import urllib.request

import urllib.request -aurllib.request module uses HTTP/1.1 and includes Connection:close header in its HTTP requests.

This function always returns an object which can work as context manager and has methods such as

  1. geturl() — return the URL of the resource retrieved, commonly used to determine if a redirect was followed(检查是否重定向)
  2. info() — return the meta-information of the page, such as headers, in the form of an email.message_from_string() instance (see Quick Reference to HTTP Headers)
  3. getcode() – return the HTTP status code of the response.
